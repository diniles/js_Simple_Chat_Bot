Determining if a number is even or odd

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create an un-named (also known as an anonymous) JavaScript function that receives an integer as input and returns a string which summarizes if the input number is even or odd. If the number is even, return the string 'Even'. If the number is odd, return the string 'Odd'.
Sample Input:
2

Sample Output:
Even


Sample Input:
3

Sample Output:
Odd



Memory limit: 256 MB
Time limit: 15 seconds
